# Intensive Science Computer Website

Welcome to the **Intensive Science Computer Website** project.

This website was developed to provide and display various information related to academic study plans, especially for the **Intensive Science and Computer Program**.
The platform is designed to help students easily access important details, schedules, announcements, and other educational resources in one place.

---

## 📚 Features

* Study plan information
* Academic schedules
* Program announcements
* Student resources
* Easy-to-use web interface

---

## 🚀 Purpose

The purpose of this project is to improve accessibility to educational information and create a centralized platform for students and staff.
